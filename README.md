# Strava Year-End Running Summary

A Flask web application that connects to your Strava account to generate year-end running summaries with AI-powered insights.

## Features

- **Strava Integration**: Connect your Strava account to fetch running activities
- **Year-End Summary**: View your 2025 running statistics in a clean table format
- **ChatGPT Integration**: Get AI-powered analysis and coaching insights
- **Copy-to-Clipboard**: Export your data with pre-written prompts for ChatGPT analysis
- **IST Timezone Support**: All timestamps converted to Indian Standard Time
- **Render Deployment**: Ready for deployment on Render.com

## Environment Variables

Set these environment variables in your Render dashboard:

- `STRAVA_CLIENT_ID`: Your Strava application client ID
- `STRAVA_CLIENT_SECRET`: Your Strava application client secret
- `OPENAI_API_KEY`: Your OpenAI API key for ChatGPT integration
- `FLASK_SECRET_KEY`: Auto-generated by Render
- `REDIRECT_URI`: Set to `https://your-app-name.onrender.com/callback`

## Setup

1. **Create Strava App**:
   - Go to [Strava API Settings](https://www.strava.com/settings/api)
   - Create a new application
   - Set Authorization Callback Domain to `your-app-name.onrender.com`

2. **Deploy to Render**:
   - Connect this repository to Render
   - Set the environment variables above
   - Deploy!

3. **Update Strava Callback**:
   - After deployment, update your Strava app's Authorization Callback Domain to match your Render URL

## Usage

1. Visit your deployed app
2. Click "Login with Strava"
3. Authorize the application
4. View your 2025 running statistics
5. Use the copy buttons to export data for ChatGPT analysis

## ChatGPT Prompts

The app provides 4 pre-written prompts:
1. **Basic Analysis**: Performance trends and recommendations
2. **Visual Poster**: Spotify Wrapped-style infographic
3. **Coaching Analysis**: Professional running coach insights
4. **Social Media**: Platform-specific shareable content
